# AI Hackathon System

CodeMentor AI's AI Hackathon module is a fully integrated, intelligent event management and evaluation system.

It supports:

- Event creation
- Team participation
- Project submission
- AI-based scoring
- Manual judging
- Leaderboards
- Rewards & certificates
- Payment integration (entry fee)

---

## 🚀 Why AI Hackathons?

Traditional hackathons rely heavily on manual judging.

CodeMentor AI enhances the system with:

- AI-assisted evaluation
- Code logic scoring
- Similarity detection
- Skill mapping
- Performance ranking
