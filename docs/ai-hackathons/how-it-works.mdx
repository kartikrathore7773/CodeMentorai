# How It Works

## Step 1: Hackathon Creation

Admin creates:

- Title
- Description
- Entry Fee
- Deadline
- Evaluation Criteria

---

## Step 2: Registration

Users:

- Pay entry fee (if required)
- Join or create team
- Submit required details

---

## Step 3: Submission

Teams submit:

- GitHub link
- Demo link
- Documentation
- AI summary (optional)

---

## Step 4: Evaluation

System performs:

- Code similarity check
- Basic logic scoring
- Feature detection

Judges provide:

- Innovation score
- UI score
- Impact score

---

## Step 5: Leaderboard

Scores combined:
AI Score + Judge Score = Final Rank
