# Mentor Dashboard

The Mentor Dashboard allows mentors to manage:

- Student subscriptions
- Google Meet sessions
- Telegram group access
- Progress tracking
- Session notes
- Performance reports

---

## 📊 Dashboard Features

### 1️⃣ Student Management

- View subscribed users
- Track plan type
- Session history

### 2️⃣ Session Scheduling

- Create Google Meet links
- Set session time
- Notify participants

### 3️⃣ Telegram Access Control

- Generate invite links
- Manage premium members
- Remove expired members

### 4️⃣ Performance Tracking

- Track progress
- Upload feedback
- Assign tasks

---

## 🔐 Role-Based Access

- Mentor
- Admin
- Premium User

Each role has restricted dashboard permissions.
