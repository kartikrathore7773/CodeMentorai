# Content Moderation & Quality Control

We maintain a multi-layer moderation system:

## Automated Checks

- Plagiarism detection
- AI hallucination detection
- Code validation
- Toxicity filtering
- Fact consistency check

---

## Manual Review Layer

- Technical validation
- Code correctness check
- Job authenticity verification
- Referral authenticity check

---

## Quality Score System

Each blog receives:

- Technical Depth Score
- Clarity Score
- SEO Score
- Trust Index
- Community Rating
